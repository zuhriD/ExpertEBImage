EBImage

EBImage menyediakan fungsi dan tujuan umum untuk pemrosesan dan analisis gambar. Dalam konteks pengujian seluler berbasis mikroskop (high-throughput), EBImage menawarkan alat untuk menyegmentasikan sel dan mengekstrak deskriptor seluler kuantitatif. Hal ini memungkinkan otomatisasi tugas tersebut menggunakan bahasa pemrograman R dan memfasilitasi penggunaan alat lain di lingkungan R untuk pemrosesan sinyal, pemodelan statistik, pembelajaran mesin, dan visualisasi dengan data gambar.

Menampilkan Gambar

Pada dasarnya nya fungsi dari (“EBImage”) pada package R adalah mencakup membaca,menulis dan menampilkan gambar,Disini gambar dibaca menggunakan fungsi “ReadImage” dengan cara mengambil nama file atau URL gambar sebagai masukannya.baiklah berikut cara menampilkan gambar menggunakan fungsi display()

Menggunakan display()

# menampilkan gambar

Image <- readImage('masjid.jpg')
display(Image)

Paket R (“EBImage”) saat ini mendukung tiga format file gamabr yaitu jpeg,png,tiff dan untuk Format gambar tambahan dapat diimpor menggunakan R GitHubPkg (“aoles / RBioFormats”), yang menambahkan dukungan untuk berbagai format file termasuk data gambar mikroskopis dan metadata.selain menggunakan fungsi display(image).

Menggunakan method=“browser”

# menampilkan gambar

img = readImage("masjid.jpg")
display(img, method="browser")

Memanipulasi gambar

Dengan array numerik gambar dapat dengan mudah dimanipulasi oleh salah satu operator aritmatika dalam R.Misalnya,kita dapat menghasilkan gambar negatif hanya dengan mengurangi nilai gambar dari maksimumnya. 

img_neg = max(img) - img
display( img_neg )

Kita juga bisa meningkatkan kecerahan pada gambar dengan melalui penambahan,menyesuaikan kontras dengan perkalian dan menerapkan koreksi gamma melalui eksponen 

img_comb = combine(
  img,
  img + 0.3,
  img * 2,
  img ^ 0.5
)

display(img_comb, all=TRUE)

Pada percobaan ini kita mencoba menggabungkan beberapa gambar kedalam satu objek gambar dengan multi bingkai.Kemudian kita juga dapat memotong dan membuat gambar thresholding dengan operasi matriks standar 

colorMode(Image) <- Color
display(Image[50:600, 2:300,])

Untuk transposisi gambar,dapat menggunakan fungsi transpose dalam pake R’s R (base") dengan fungsi t.hal ini juga dapat berfungsi pada gambar berwarna dan multiframe dengan cara menukar dimensi spasialnya 

img_t = transpose(img)
display( img_t )

Transformasi Spasial

Pada percobaan sebelumnya kita telah mengetahui jenis-jenis transformasi spasial dan transposisi,namun jenis-jenisnya masih banyak lagi misalnya translasi,rotasi,refleksi,dan penskalaan.Translasi memindahkan bidang gambar dengan vektor dua dimensi yang ditentukan sedemikian rupa sehingga piksel yang berakhir di luar wilayah gambar akan dipotong,dan piksel yang masuk ke wilayah gambar disetel ke latar belakang

Translasi gambar

img_translate = translate(img, c(100,-50))
display(img_translate)

Merotasi gambar

Warna latar belakang dapat disetel menggunakan argumen bg.co; yang umum untuk digunakan pada semua fungsi transformasi spasial yang relevan.Sementara default disini mengatur nilai piksel latar belakang menjadi ke nol dengan warna hitam.Berikut penggunaan rotate dimana rotate ini sendiri berfungsi untuk memutar gambar searah jarum jam sesuai dengan sudut yang diberikan 

img_rotate = rotate(img, 30, bg.col = "white")
display(img_rotate)

Mengubah ukuran gambar

Untuk menskalakan gambar ke dimensi yang diinginkan,dapat mengunakan pengubahan ukuran.Jika hanya memberikan salah satu dari lebar atau tinggi,pada dimensi lainya secara otomatis dihitung dengan tetap menggunakan aspek rasio asli. 

img_resize = resize(img, w=600, h=600)
display(img_resize )

Merefleksi gambar

Fungsi flip dan flop disini adalah merefleksikan gambar di sekitar sumbu horizontal dan vertikal masing-masing. 

img_flip = flip(img)
img_flop = flop(img)

display(combine(img_flip, img_flop), all=TRUE)

Transformasi gambar

Transformasi linear spasial diimplementasikan menggunakan transformasi affine umum.Hal ini memetakan koordinat piksel gambar px menggunakan matriks transformasi 3x2m dengan cara berikut: cbind (px, 1)% *% m 

m =  matrix(c(1, -.5, 128, 0, 1, 0), nrow=3, ncol=2)
img_affine = affine(img, m)
display( img_affine )

Memberikan filter

Dalam r Biocpkg (“EBImage”), konvolusi 2 dimensi diimplementasikan oleh fungsi filter2, dan fungsi tambahan makeBrushdapat digunakan untuk menghasilkan fungsi bobot. 

w = makeBrush(size = 31, shape = 'gaussian', sigma = 5)
plot(w[(nrow(w)+1)/2, ], ylab = "w", xlab = "", cex = 0.7)

adalah operasi kebalikannya yang memungkinkan untuk mendeteksi tepi dan mempertajam gambar. 

fhi = matrix(1, nrow = 3, ncol = 3)
fhi[2, 2] = -8
img_fhi = filter2(img, fhi)
display(img_fhi)

Teknik pemfilteran ini disediakan dalam paket R (“EBImage”) oleh fungsinya medianFilter. Pada gambar ini kita memakai noise seragam, dan merekonstruksi gambar asli dengan median filtering

Operasi morfologi Gambar biner adalah gambar yang hanya berisi dua set piksel, dengan nilai, katakanlah 0 dan 1, mewakili piksel latar belakang dan latar depan. 

shapes = readImage(system.file('images', 'shapes.png', package='EBImage'))
logo = shapes[110:512,1:130]
display(logo)

kern = makeBrush(5, shape='diamond')
display(kern, interpolate=FALSE)

logo_erode= erode(logo, kern)
logo_dilate = dilate(logo, kern)

display(combine(logo_erode, logo_dilate), all=TRUE)

Segmentasi gambar

Segmentasi gambar melakukan partisi gambar, dan biasanya digunakan untuk mengidentifikasi objek dalam gambar. Objek terhubung yang tidak bersentuhan dapat disegmentasi menggunakan fungsi tersebut bwlabel, sedangkan watershed dan propagate menggunakan algoritma yang lebih canggih dapat memisahkan objek yang saling bersentuhan. 

logo_label = bwlabel(logo)
table(logo_label)
## logo_label
##     0     1     2     3     4     5     6     7 
## 42217  1375  2012   934  1957  1135  1697  1063

Nilai piksel logo_label rentang gambar dari 0 sesuai dengan latar belakang hingga jumlah objek yang dikandungnya, yang diberikan oleh max(logo_label). Untuk menampilkan gambar kita menormalkannya ke kisaran (0,1) yang diharapkan oleh fungsi tampilan. Ini menghasilkan objek berbeda yang dirender dengan bayangan abu-abu yang berbeda. Bisa juga kita membuat gambar berwarna warni. 

max(logo_label)
## [1] 7
display( normalize(logo_label) )

display( colorLabels(logo_label) )

Memanipulasi objek

Objek dapat dihapus dari gambar tersebut dengan rmObject, yang menghapus objek dari topeng hanya dengan mengatur nilai pikselnya ke 0. Secara default, setelah penghapusan objek semua objek yang tersisa diberi label ulang sehingga ID objek tertinggi sesuai dengan jumlah objek di topeng. The reenumerate argumen dapat digunakan untuk mengubah perilaku ini dan untuk melestarikan ID objek asli. 

objects = list(
    seq.int(from = 2, to = max(logo_label), by = 2),
    seq.int(from = 1, to = max(logo_label), by = 2)
    )
logos = combine(logo_label, logo_label)
z = rmObjects(logos, objects, reenumerate=FALSE)
display(z, all=TRUE)

Segmentasi sel

Kita bisa memuat gambar inti dan badan sel. Untuk memvisualisasikan sel, kita melapisi gambar-gambar ini sebagai saluran hijau dan biru dari gambar berwarna palsu. 

nuc = readImage(system.file('images', 'nuclei.tif', package='EBImage'))
cel = readImage(system.file('images', 'cells.tif', package='EBImage'))

cells = rgbImage(green=1.5*cel, blue=nuc)
display(cells, all = TRUE)

Pertama, kita segmentasi menggunakan thresh, fillHull, bwlabel dan opening. 

nmask = thresh(nuc, w=10, h=10, offset=0.05)
nmask = opening(nmask, makeBrush(5, shape='disc'))
nmask = fillHull(nmask)
nmask = bwlabel(nmask)

display(nmask, all=TRUE)

Selanjutnya, kami menggunakan inti tersegmentasi sebagai benih dalam segmentasi Voronoi dari sitoplasma. 

ctmask = opening(cel>0.1, makeBrush(5, shape='disc'))
cmask = propagate(cel, seeds=nmask, mask=ctmask)

display(ctmask, all=TRUE)

Untuk memvisualisasikan segmentasi kita pada yang kita gunakan paintObject. 

segmented = paintObjects(cmask, cells, col='#ff00ff')
segmented = paintObjects(nmask, segmented, col='#ffff00')

display(segmented, all=TRUE)

Pemrosesan Gambar dan Representasi Spektral Sinyal

Kita coba untuk menginterogasi gambar dibawah ini. 

# panggil package
library(EBImage)

# read image
image <- readImage('masjid.jpg')
display(image, method = "raster", all = TRUE)

# Intensity Histogram
hist(image)

# Generate negative image
img_neg <- max(image) - image
display( img_neg )

# dim: 633 1000    3

# crop and threshold
img_crop <- image  
img_thresh <- img_crop > 0.55
display(img_thresh)

# Low-pass Filter/smooth
w <- makeBrush(size = 31, shape = 'gaussian', sigma = 5)
img_flo <- filter2(image, w)
display(img_flo)

# High-pass filtering Laplace Filter/Edge detection
fhi <- matrix(1, nrow = 3, ncol = 3)
fhi[2, 2] <- -7
img_fhi <- filter2(image, fhi)
display(img_fhi)

# Denoising
img_median <- medianFilter(image, size=5, cacheSize=20000)
display(img_median)

# Segmentation
grayimage <- channel(image,"gray")
threshold <- otsu(grayimage)
CC_th <- combine( mapply(function(frame, th) frame > th, 
                         getFrames(image), 
                         threshold, SIMPLIFY=FALSE) )
display(CC_th, all=TRUE)

disc <- makeBrush(51, "disc")
disc <- disc/sum(disc)
offset <- 0.0001
CC_bg <- filter2(image, disc)
CC_th <- image < CC_bg + offset
display(CC_th, all=TRUE)

# Watershed segmentation
nmask <- watershed(distmap(CC_th), tolerance=5, ext=20)
display(colorLabels(nmask), all=TRUE)

# Voronoi image tesselation
voronoiExamp <- propagate(x = nmask, seeds = nmask, lambda = 0.001)
voronoiPaint <- colorLabels (voronoiExamp)
display(voronoiPaint, all=TRUE)

CC <- rgbImage(green=2.0*CC_bg, blue=img_median, red=voronoiPaint[ , , , 1])
display(CC, all = TRUE)

ctmask = opening(img_fhi<0.6, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

ctmask = opening(img_fhi>0.1, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

segmented = paintObjects(channel(cmask,"gray"), image, col='#ff00ff')
segmented = paintObjects(channel(ctmask,"gray"), segmented, col='#ffff00')
display(segmented, all=TRUE)
